Вопрос: Какие типы ламп преобладают?
├── 100% LED (10 баллов)
│   └── Рекомендация: Поддерживать текущий уровень
├── В основном LED (8 баллов)
│   └── Рекомендация: Заменить оставшиеся не-LED лампы
├── Смешанные (5 баллов)
│   └── Переход к: Частота использования освещения
│       ├── Экономное использование → Умные датчики движения
│       └── Частое использование → Полная замена на LED + датчики
└── В основном лампы накаливания (2 балла)
    └── Переход к: Частота использования освещения
        ├── Экономное использование → СРОЧНАЯ ЗАМЕНА НА LED
        └── Частое использование → СРОЧНАЯ ЗАМЕНА + умное управление

Вопрос: Класс энергоэффективности основной техники?
├── A+++/A++ (10 баллов)
│   └── Рекомендация: Плановое обновление через 5-7 лет
├── A/A+ (8 баллов)
│   └── Рекомендация: При замене выбирать A+++
├── Смешанные B/C (5 баллов)
│   └── Переход к: Режимы ожидания техники
│       ├── Полное отключение → Приоритетная замена самой старой техники
│       └── Режим ожидания → Умные розетки + замена техники
└── Старая техника D/G (2 балла)
    └── Переход к: Режимы ожидания техники
        ├── Полное отключение → КРИТИЧЕСКАЯ ЗАМЕНА энергоемкой техники
        └── Режим ожидания → Умные розетки + СРОЧНАЯ ЗАМЕНА техники

Вопрос: Какая система кондиционирования используется?
├── Инверторный A+++ (10 баллов)
│   └── Рекомендация: Оптимальные настройки (24-25°C)
├── Обычный кондиционер класса A (7 баллов)
│   └── Рекомендация: Обслуживание + программируемый термостат
├── Старый кондиционер низкого класса (3 балла)
│   └── Переход к: Частота использования кондиционера
│       ├── Экономное использование → Обслуживание + замена при поломке
│       └── Частое использование → СРОЧНАЯ ЗАМЕНА на инверторный
└── Нет кондиционера, используем вентиляторы (8 баллов)
    └── Рекомендация: Умные таймеры для вентиляторов

Вопрос: Как нагревается вода?
├── Газовый или тепловой насос (10 баллов)
│   └── Рекомендация: Оптимально, утепление труб
├── Электрический бойлер с изоляцией (6 баллов)
│   └── Рекомендация: Дополнительное утепление + настройки температуры
├── Проточный электрический нагреватель (4 балла)
│   └── Переход к: Объем потребления горячей воды
│       ├── До 50 л/чел → Оптимизация использования
│       └── Более 100 л/чел → ЗАМЕНА на накопительный бойлер
└── Старый электрический бойлер (2 балла)
    └── Переход к: Объем потребления горячей воды
        ├── Любой объем → КРИТИЧЕСКАЯ ЗАМЕНА на современный бойлер

Вопрос: Какая у вас плита и духовка?
├── Индукционная плита + конвекционная духовка A++ (10 баллов)
│   └── Рекомендация: Использовать крышки при готовке
├── Стеклокерамическая плита + обычная духовка A (7 баллов)
│   └── Рекомендация: При замене выбирать индукционную
├── Электрические спирали + старая духовка (3 балла)
│   └── Рекомендация: ПРИОРИТЕТНАЯ ЗАМЕНА на индукционную плиту
└── Газовая плита (9 баллов)
    └── Рекомендация: Оптимально для экономии электроэнергии

Вопрос: Какое количество электроники постоянно работает?
├── Минимум, только необходимое (10 баллов)
│   └── Рекомендация: Поддерживать текущий уровень
├── Умеренное количество с умным управлением (8 баллов)
│   └── Рекомендация: Оптимизировать настройки сна
├── Много устройств, некоторые постоянно включены (5 баллов)
│   └── Переход к: Наличие игрового оборудования
│       ├── Нет игрового → Умные розетки + настройки энергосбережения
│       └── Есть игровое → Таймеры автоматического выключения
└── Офис или развлекательный центр (3 балла)
    └── Переход к: Наличие игрового оборудования
        ├── Нет игрового → Сетевые фильтры + политика отключения
        └── Есть игровое → КОМПЛЕКСНАЯ ОПТИМИЗАЦИЯ энергопотребления
